# 🦊 teto-flo

> 「こうしたいけどできない」をAIと話すだけで、解決の道筋を一緒に作ってくれるアプリ。

**QuestQueries** は「真面目なことを、プチ冒険として。」をコンセプトにした実験的なAIツール集です。  
teto-flo はその汎用入口として作られました。m-flo へのリスペクトも込めて。

---

## ✨ どんなアプリ？

プロンプトを書かなくていい。フレームワークを知らなくていい。  
テトに話しかけるだけで、いつの間にか**自分だけの実行計画**ができています。

```
「マラソン練習したいけど続かなくて…」
    ↓ テトが友達感覚で深掘り
「じゃあこんな感じでどう？」（構成案を提示）
    ↓ OK を返すだけ
プレビューカードが表示 → 確認して「HTMLを保存」
```

フローチャートが必須ではありません。相談の内容によって、フローチャート・ルーティン・箇条書きなど、テトが最適な形式を提案します。

---

## 🚀 使い方

### 1. ファイルを開く

`index.html` をブラウザでダブルクリックするだけ。インストール不要。

### 2. APIキーを設定する

初回起動時にモーダルが表示されます。  
[Anthropic Console](https://console.anthropic.com/settings/keys) で取得した APIキー（`sk-ant-...`）を入力してください。

> ⚠️ **使用量の上限を設定することを強く推奨します。**  
> → [使用量上限の設定はこちら](https://console.anthropic.com/settings/limits)  
> キーはあなた自身の Anthropic アカウントに紐づいており、課金もあなたのアカウントで発生します。  
> 入力したキーはこのブラウザのみに保存され、外部には送信されません。

### 3. テトと話す

自然な言葉で話しかけてください。  
テトが 2〜3 往復で深掘りし、構成案を提示します。  
「いいね」「これで」と返すと、チャット内にプレビューカードが表示されます。

### 4. 確認して保存

プレビューカードの内容を確認して「⬇ HTMLを保存」ボタンを押すと、  
フローチャート付きのHTMLファイルがダウンロードされます。  
ダブルクリックで開けばいつでも見返せます。

## 🛠️ 技術構成

| 項目 | 内容 |
|---|---|
| 実行環境 | ブラウザのみ（サーバー不要） |
| AI | Anthropic Claude API（claude-sonnet-4） |
| 図の描画 | [Mermaid.js](https://mermaid.js.org/) v10 |
| フォント | Zen Kaku Gothic New / Space Mono（Google Fonts） |
| データ保存 | APIキーのみ localStorage に保存（会話履歴は保存しない） |

### ファイル構成

```
teto-flo/
└── index.html   # アプリ本体（1ファイル完結）
```

## 🦊 テトについて

ナウシカとテトの関係性がベース。  
無口な相棒が話しかけてきて、一緒に問題を解いていく。  
「なぜ？」とは聞かない。「どうなりたい？」から入る。  
押しつけず、でも深いところまで掘る。

### AIはパートナーである、という思想

テトは「ツール」ではなく「相棒」として設計されています。

同じ犬種でも、育てた人によって性格が違う。AIも同じです。  
テトは使う人・環境・積み重ねた対話によって、少しずつ違う反応を返します。  
それはバグではなく、パートナーとしてのAIの本質です。

> RPAやシステムは「同じ入力に同じ出力」を返します。  
> AIは「文脈・相手・環境」によって変わります。  
> その違いを理解した上で、テトと付き合ってください。
